# Fiji 汉化版（最新版本独家）

## 简介

本仓库提供 **Fiji（Fiji Is Just ImageJ）** 的中文汉化版本。Fiji 是一个基于 ImageJ 的强大的图像处理软件，广泛应用于生物医学图像分析领域。

本汉化版对 Fiji 的界面进行了全面的中文化，方便中文用户使用。

## 关于 Fiji

Fiji 是一个"开箱即用"的 ImageJ 发行版，预装了大量常用的插件，包括：

- 生物医学图像分析工具
- 细胞分割与追踪
- 图像配准与拼接
- 三维可视化与分析
- 机器学习分割（如 Trainable Weka Segmentation）

官方网站：https://fiji.sc/
